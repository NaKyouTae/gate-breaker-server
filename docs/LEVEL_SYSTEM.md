# Gate Breaker - 레벨 시스템 문서

## 1. 레벨 정책

- 레벨 상한: **무제한**
- 현재 서버 로직(`battle.service.ts`)도 레벨 캡 없이 `while` 루프로 레벨업 처리

## 2. 경험치 계산식

- 다음 레벨 필요 경험치:
- `requiredExp(level) = floor(100 * level^1.5)`
- 레벨 시작 누적 경험치(해당 레벨 달성 총 필요치):
- `cumulativeExp(level) = sum(requiredExp(k)), k=1..(level-1)`

해석:
- 예) 레벨 30에서 31로 가려면 `requiredExp(30) = 16,431`
- 예) 레벨 30 시작 시점 누적 경험치는 `189,018`

## 5. 운영 메모

- 현재 `levelup.exp_table.base = 100`이므로 성장 곡선이 가파른 편
- 레벨 구간별 체감 조정이 필요하면 `/config/levelup/exp_table`에서 `base`를 조정
- 스탯 성장치(HP/MP/공/방)는 `/config/levelup/stat_growth`로 별도 조정 가능

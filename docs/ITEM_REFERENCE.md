# Gate Breaker - 아이템 정보 문서

## 문서 목적

- 아이템 도감/상점/인벤토리 UI의 기준 레퍼런스
- 포함 항목: 착용 레벨, 상점 판매가/구매가, 기본 능력치, 속성 태그
- 기준: 현재 시드 아이템 + Lv 1~100 확장안 아이템

## 1. 착용 레벨 정책

현재 DB에는 아이템 착용 레벨 필드가 없으므로, 도감/기획 기준값으로 관리한다.

- 장비 아이템: `requiredLevel` 사용
- 재료/소비 아이템: 착용 레벨 없음(`-`)

## 4. 속성 태그 정의

- `공격형`: 공격력 위주 성장
- `방어형`: 방어력/HP 위주 성장
- `균형형`: 공격/방어가 함께 상승
- `생존형`: HP/감쇠 중심으로 전투 지속력 강화
- `회복`: 전투 중 회복 목적 소비 아이템
- `기본재료`: 몬스터 드랍 기반 일반 재료
- `희귀`: 상점 미판매 또는 보스 중심 드랍

## 5. 도감/상점 연동 권장 필드

- 기본 필드: `name`, `type`, `rarity`, `requiredLevel`, `baseAttack`, `baseDefense`, `baseHp`, `sellPrice`, `buyPrice`, `tags`
- 표시 필드: `description`, `source`, `dropMonsters[]`, `isShopAvailable`
- 정렬 키: `requiredLevel ASC`, `rarity DESC`, `type ASC`

## 6. 운영 메모

- 착용 레벨(`requiredLevel`)은 현재 스키마 미포함이므로 도입 시 마이그레이션 필요
- 상점 구매 가능 여부는 현재 `buyPrice >= 0` (null이 아닌 경우)으로 판단
- 확장안 아이템은 [ITEM_CONCEPT.md](/Users/nakyutae/personal/git/gate-breaker/gate-breaker-server/docs/ITEM_CONCEPT.md) 기준

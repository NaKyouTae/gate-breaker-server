# Gate Breaker - 몬스터 도감 데이터 문서

## 문서 목적

- 추후 몬스터 도감 UI/DB의 기준 데이터로 사용
- 현재 기준: `prisma/seed.ts` 데이터
- 포함 항목: 몬스터 레벨, HP, EXP, 골드, 드랍 아이템(장비/재료/소비)

## 1. 도감용 몬스터 레벨 규칙

현재 `monster` 테이블에는 레벨 필드가 없으므로, 도감에서 사용할 레벨은 아래 규칙으로 산정한다.

- 일반 몬스터 레벨: `floor((던전 minLevel + 던전 maxLevel) / 2)`
- 보스 몬스터 레벨: `던전 maxLevel`

## 4. 도감 연동 시 권장 필드

- 기본: `id`, `name`, `level`, `isBoss`, `hp`, `attack`, `defense`, `expReward`, `goldReward`
- 관계: `dungeon`, `drops[]`
- 표시 확장(선택): `lore`, `habitat`, `weakness`, `firstSeenAt`

# Gate Breaker - 게임 기획 문서

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | **Gate Breaker** |
| 장르 | 채팅형 텍스트 RPG + 아이템 파밍 |
| 컨셉 참고 | 나혼자만 레벨업 / 내가 키운 S급들 |
| 플랫폼 | 웹 (PC / 모바일 웹) |
| 핵심 재미 | 던전 → 아이템 드랍 → 강화 → 더 강한 던전 |

## 2. 게임 루프

```
던전 입장
    ↓
몬스터 전투 (턴제)
    ↓
아이템 드랍
    ↓
장비 장착 / 강화
    ↓
더 강한 던전
```

## 3. 플레이어 시스템

### 기본 스탯

| 스탯 | 설명 |
|------|------|
| 레벨 | 경험치 획득으로 레벨업 |
| HP | 체력 |
| MP | 마나 (스킬 사용) |
| 공격력 | 기본 공격 데미지 |
| 방어력 | 받는 데미지 감소 |
| 치명타율 | 크리티컬 확률 |

### 장비 슬롯 (6개)

| 슬롯 | 주요 옵션 |
|------|-----------|
| 무기 | 공격력 |
| 갑옷 | 방어력, HP |
| 장갑 | 공격력, 치명타 |
| 신발 | 회피, 속도 |
| 반지 | 특수 옵션 |
| 목걸이 | 특수 옵션 |

### 아이템 등급

| 등급 | 색상 | 드랍 확률 |
|------|------|-----------|
| Common | 회색 | 60% |
| Rare | 파랑 | 25% |
| Epic | 보라 | 10% |
| Legendary | 주황 | 4% |
| Mythic | 빨강 | 1% |

### 아이템 옵션 예시

- 공격력 +12
- 치명타 +5%
- 흡혈 +3%
- HP +50
- 방어력 +8

## 4. 강화 시스템

### 강화 확률표

| 단계 | 성공률 | 실패 시 |
|------|--------|---------|
| +0 → +1 | 100% | - |
| +1 → +2 | 90% | 유지 |
| +2 → +3 | 80% | 유지 |
| +3 → +4 | 65% | -1 하락 |
| +4 → +5 | 50% | -1 하락 |
| +5 → +6 | 35% | -2 하락 |
| +6 → +7 | 20% | 파괴 가능 |

### 강화 비용

- **골드**: 강화 단계에 비례하여 증가
- **강화석**: 던전 드랍 아이템, 단계별 필요 수량 증가

## 5. 던전 시스템

| 던전 | 권장 레벨 | 몬스터 | 주요 드랍 |
|------|-----------|--------|-----------|
| 슬라임 숲 | 1~5 | 슬라임, 슬라임 킹 | 녹슨 검, 가죽 갑옷, 슬라임 점액 |
| 고블린 동굴 | 5~10 | 고블린, 고블린 샤먼 | 고블린 검, 고블린 방패 |
| 오크 요새 | 10~20 | 오크, 오크 대장 | 전투 도끼, 강철 갑옷 |
| 언데드 묘지 | 20~30 | 스켈레톤, 리치 | 저주받은 검, 영혼석 |
| 드래곤 둥지 | 30~50 | 와이번, 드래곤 | Legendary 장비 |

### 던전 보상

- 몬스터 처치 시 경험치 + 골드
- 드랍 테이블 기반 아이템 획득
- 보스 처치 시 추가 보상

## 6. 전투 시스템

### 턴제 전투

```
플레이어 턴
  → 공격 / 스킬 / 아이템 사용 / 도망
      ↓
몬스터 턴
  → 공격 / 특수 스킬
      ↓
반복
      ↓
승리 → 경험치 + 골드 + 아이템 드랍
패배 → 마을로 귀환 (골드 일부 손실)
```

### 데미지 계산

```
기본 데미지 = (공격력 + 무기 공격력) - 상대 방어력 + random(-3, +3)
치명타 데미지 = 기본 데미지 × 1.5
최소 데미지 = 1 (항상 1 이상)
```

### 전투 화면 예시 (채팅형)

```
[슬라임 숲 - 1층]

슬라임이 나타났다!

> Kevin의 공격!
  슬라임에게 14 데미지!

> 슬라임의 반격!
  Kevin에게 3 데미지!

> Kevin의 공격!
  치명타! 슬라임에게 21 데미지!

> 슬라임을 처치했다!
  경험치 +15, 골드 +8
  [슬라임 점액]을 획득했다!

[공격] [스킬] [아이템] [도망]
```

## 7. 경제 시스템

| 기능 | 설명 |
|------|------|
| 상점 판매 | 아이템을 NPC에게 골드로 판매 |
| 상점 구매 | 기본 장비, 포션 구매 |
| 교환 | 유저 간 아이템 교환 (향후) |
| 경매장 | 아이템 경매 (향후) |

### 골드 획득 경로

- 몬스터 처치
- 아이템 판매
- 던전 클리어 보상
- 퀘스트 보상 (향후)

## 10. DB 설계

### ERD

```
[user] 1──N [inventory] N──1 [item]
  │
  │ 1──N [equipment]
  │
  │ 1──N [battle_log] N──1 [dungeon]
  │                          │
  │                     1──N [monster]
  │                          │
  │                     1──N [drop_table] N──1 [item]
```

### 테이블 정의

#### user (유저)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| nickname | VARCHAR(20) | 닉네임 |
| level | INT | 레벨 (기본 1) |
| exp | INT | 경험치 |
| gold | INT | 골드 |
| hp | INT | 현재 HP |
| max_hp | INT | 최대 HP |
| mp | INT | 현재 MP |
| max_mp | INT | 최대 MP |
| attack | INT | 기본 공격력 |
| defense | INT | 기본 방어력 |
| critical_rate | FLOAT | 치명타율 |
| created_at | TIMESTAMP | 생성일 |

#### item (아이템)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| name | VARCHAR(50) | 아이템명 |
| type | ENUM | weapon, armor, glove, shoe, ring, necklace, material, consumable |
| rarity | ENUM | common, rare, epic, legendary, mythic |
| base_attack | INT | 기본 공격력 |
| base_defense | INT | 기본 방어력 |
| base_hp | INT | 기본 HP |
| description | TEXT | 설명 |
| sell_price | INT | 판매 가격 |

#### inventory (인벤토리)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| user_id | UUID | FK → user |
| item_id | UUID | FK → item |
| enhance_level | INT | 강화 단계 (기본 0) |
| is_equipped | BOOLEAN | 장착 여부 |
| equipped_slot | VARCHAR | 장착 슬롯 |
| quantity | INT | 수량 (재료/소모품) |

#### dungeon (던전)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| name | VARCHAR(50) | 던전명 |
| min_level | INT | 최소 레벨 |
| max_level | INT | 최대 레벨 |
| reward_gold_min | INT | 최소 골드 보상 |
| reward_gold_max | INT | 최대 골드 보상 |
| reward_exp | INT | 경험치 보상 |

#### monster (몬스터)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| name | VARCHAR(50) | 몬스터명 |
| dungeon_id | UUID | FK → dungeon |
| hp | INT | HP |
| attack | INT | 공격력 |
| defense | INT | 방어력 |
| exp_reward | INT | 처치 경험치 |
| gold_reward | INT | 처치 골드 |
| is_boss | BOOLEAN | 보스 여부 |

#### drop_table (드랍 테이블)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| monster_id | UUID | FK → monster |
| item_id | UUID | FK → item |
| drop_rate | FLOAT | 드랍 확률 (0~1) |

#### battle_log (전투 로그)

| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | UUID | PK |
| user_id | UUID | FK → user |
| dungeon_id | UUID | FK → dungeon |
| result | ENUM | victory, defeat, escape |
| gold_earned | INT | 획득 골드 |
| exp_earned | INT | 획득 경험치 |
| created_at | TIMESTAMP | 전투 일시 |

## 11. 향후 확장 기능

| 기능 | 우선순위 | 설명 |
|------|----------|------|
| 스킬 시스템 | 높음 | 직업별 스킬 트리 |
| 랭킹 | 높음 | 레벨/전투력 순위 |
| 길드 | 중간 | 길드 생성, 길드 던전 |
| PVP | 중간 | 유저 간 대전 |
| 경매장 | 중간 | 유저 간 아이템 거래 |
| 월드보스 | 낮음 | 다수 유저 협동 레이드 |
| 펫 시스템 | 낮음 | 동반 펫 |
| 시즌제 | 낮음 | 시즌별 랭킹 초기화 |
